# XMedify – Healthcare Management Platform

## Overview

XMedify is a healthcare management platform built using React.js that helps users discover hospitals, compare healthcare facilities, and manage appointment bookings efficiently. The application provides a seamless user experience through responsive design, dynamic search functionality, and real-time healthcare information.

## Features

* Hospital search and discovery
* State-wise and city-wise hospital filtering
* Appointment booking workflow
* Real-time healthcare data integration
* Responsive user interface
* Hospital ratings and information display
* Session persistence using localStorage
* Reusable React component architecture

## Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Bootstrap

### APIs

* Healthcare Data APIs
* REST API Integration

### Deployment
* Vercel
